VGG Image Annotator (VIA)

Abhishek Dutta, Ankush Gupta, and Andrew Zisserman

Overview

VGG Image Annotator (VIA) is an open source project developed at the Visual Geometry Group and released under the BSD-2 clause license. With this standalone application, you can define regions in an image and create a textual description of those regions. Such image regions and descriptions are useful for supervised training of learning algorithms.

Features:

  • based solely on HTML, CSS and Javascript (no dependecies on any javascript libraries)
  • can be used offline (complete application packaged in a single html file of size < 200KB)
  • requires nothing more than a modern web browser (tested on firefox and chrome)
  • supports following region shapes: rectangle, circle, ellipse, polygon
  • supports multiple attributes for each image region
  • import (and export) of region data from (to) text file in csv and json format
  • hundreds of images can be loaded and annotated without any performance degradation

Getting Started

  1. Visit the application website (or download a copy if you wish to use it offline)
  2. Click [Image → Load or Add Image] in the top menu bar to load a set of images that you wish to annotate.
  3. Press n (or p) to navigate through the loaded images.
  4. Define some region attributes by pressing the plus sign (+) in the Region Attributes section (left side). You should be able to see newly defined attribute.
  5. In the Select Region Shape section (top left side), click the rectangular shape
  6. On the image area, keep pressing the right click button as you drag the mouse cursor. This will draw a rectangular region on the image.
  7. Using your mouse, press a single click inside this new rectangular area to select this newly created region.
  8. In the Region Attributes area (left side), you will see "[click to set value]" link beside the attribute name created in step 4. Click on this link.
  9. Type a value for this attribute and click 'Save' (or just press Enter). This new value will now be assigned to this rectangular region.
  10. To download the annotated region data, click [Annotation → Save as CSV] in the top menu bar. This will download a text file containing region shape and attribute data.
  11. Next time, you can start from the point your left by first loading the images and then importing the CSV file (downloaded in step 10) by clicking [Annotation → Import].
  12. You can also save the attributes names as a text file by clicking [Attributes → Save attributes]
